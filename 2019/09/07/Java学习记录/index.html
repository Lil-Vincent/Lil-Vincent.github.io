    Java与c++的不同

  1. Java跨平台c++不跨平台
  2. Java语言中不存在全局变量或全局函数

    C++兼具面向对象和面向过程变成的特点，可以定义全局变量和全局函数。

  3. Java没有指针

    C/C++语言相比，Java语言中没有指针的概念，这有效防止了C/C++语言中操作指针可能引起的系统问题，从而使程序变得更加安全。

  4. Java语言不支持多重继承

    但是Java引入了接口的概念，可以同时实现多个接口。由于接口也具有多态特性，因此在Java语言中可以通过实现多个接口来实现与C++语言中多重继承类似的目的。

  5. Java语言提供了垃圾回收器来实现垃圾的自动回收，不需要程序显式地管理内存的分配。
  6. 6.C++语言支持运算符重载，而Java语言不支持运算符重载。

面向对象基础

  • 定义”人”为类(class),”小明”，”小红”，”小军”为类中的的实例(instance)

class和instance

  • class是一种对象模板，定义创建某种实例。因此class本身是一种数据类型
  • instance是根据class创建的实例，可以创建多个实例，但每个实例可能各自属性不同
定义class
  • 创建一个类并取一个名字，就是定义一个class
  • 一个class包含多个字段(field),用来描述类的特征。Preson类的字段是String命名为name和int命名为age。同理Book也是如此。
创建实例
  • 根据模板class，用new操作符创建实例。
  • 创建实例需要定义一个引用类型的变量来指向这个实例。

Person ming = new Person();

注意1：此处创建一个Person类型的实例，并通过变量ming指向它

注意2：区分Person ming是定义Person类型变量ming,而new Person()是创建Person实例。

总结
  1. class是一个模板，instance是按照模板创建的实例
  2. class一个类，field(字段)用来描述类的属性
  3. 创建一个实例，需要通过创建一个该类类型的变量，该变量指向这个创建的实例
  4. 有了变量指向创建的实例就可以对class进行操作了。

方法名相同，参数的类型或个数或类型与个数都不同，称为方法重载。

多态
  1. 同一方法调用，由于对象不同 可会有不同的行为
  2. 多态是方法的多态，不是属性的多态
  3. 多态存在3个条件(继承，方法重写，父类引用指向子类对象）

父类引用指向子类对象后，用该父类引用子类重写的方法，此时多态就出现了。
