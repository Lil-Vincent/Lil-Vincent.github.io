博客搭建

  • 第一步：安装部署hexo

建立一个博客文件夹,右键gitbash hear

  • 第二步:初始化博客文件夹

    在博客文件下面右键git b输入

    type: git
    repo: https://github.com/Lil-Vincent/Lil-Vincent.github.io.git
    branch: master

注意
：此时可能上传失败，需要再次输入指令重新部署一下hexo

npm install hexo-deployer-git --save
